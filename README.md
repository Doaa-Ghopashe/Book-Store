# Book-Store
This project has two distinct sides:

User Side:
- Read books
- Rate books
- Review books
- View information about authors and their books
  
Admin Side:
- Add books
- Add authors
- Add categories
- Edit books
- Edit authors
- Edit categories
- Remove books
- Remove authors
- Remove categories
  
## Using
1. Angular
2. Node JS
3. MongoDB
4. Postman
